Redesign this Laravel Livewire app into a polished cybersecurity-themed web application.

Project purpose:
This is not a generic starter kit. It is a Secure Login Monitoring System with failed login detection, temporary account locking, and login activity logs.

Design goals:
- dark modern UI
- professional cybersecurity feel
- clean, premium, minimal
- strong visual identity
- not generic Laravel starter kit style
- responsive and polished
- use Tailwind/Flux components where possible
- keep implementation realistic for Laravel Livewire

Pages to redesign:
1. Login page
2. Dashboard
3. Security settings page

Login page requirements:
- replace generic title with a security-focused heading
- add short security message like “Multiple failed attempts may temporarily lock your account”
- improve spacing, typography, card styling, button styling
- add a subtle security visual feel (shield/lock motif, soft glow, modern panel design)
- keep the form usable and professional
- do not overdesign

Dashboard requirements:
- convert placeholder dashboard into a Security Dashboard
- include cards for:
  - successful logins
  - failed login attempts
  - locked accounts
  - recent login activity
- include a recent security logs table
- make it look like a real monitoring app

Security settings page:
- redesign to match the new style
- make profile/security sections feel consistent with the cybersecurity theme

Workflow:
1. First propose a UI redesign plan
2. Then redesign the login page
3. Then redesign the dashboard
4. Then redesign the security settings page
5. Keep all changes aligned with the current Laravel Livewire structure
6. Do not break the existing auth flow
